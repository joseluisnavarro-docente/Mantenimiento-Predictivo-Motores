# Mantenimiento Predictivo de Motores con IA

Este proyecto utiliza un microcontrolador Arduino y un acelerómetro para detectar fallas mecánicas en motores en tiempo real usando Inteligencia Artificial (TinyML).

## Hardware Utilizado
* Arduino (especifica tu modelo, ej. Nano 33 BLE o ESP32)
* Acelerómetro (ej. MPU6050)
* Módulo Relé de 5V (para el paro de emergencia del motor)

## Cómo usar este proyecto
1. Descarga el archivo `.zip` de la librería e instálalo en tu Arduino IDE.
2. Abre el archivo `.ino` y súbelo a tu placa.
3. Conecta el relé al Pin X para activar la protección automática.
